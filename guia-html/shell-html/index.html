Guia Shell Script

1. Introdução à Linguagem

Descrição: Shell Script é uma linguagem de script usada para automatizar tarefas no terminal ou linha de comando. É amplamente utilizado em sistemas Unix/Linux.

Exemplo 1: Primeiro programa em Shell Script.

#!/bin/bash
echo "Olá, mundo!"

Exemplo 2: Execução do script.

# Salve o arquivo como script.sh
# Torne-o executável:
chmod +x script.sh

# Execute:
./script.sh
2. Fundamentos

Descrição: Os fundamentos incluem variáveis, comentários e operações básicas.

Exemplo 1: Declaração de variáveis.

#!/bin/bash
nome="Shell Script"
echo "Estou aprendendo $nome"

Exemplo 2: Comentários e operações matemáticas.

#!/bin/bash
# Isso é um comentário
numero1=10
numero2=5
soma=$((numero1 + numero2))
echo "A soma é: $soma"
3. Controle de Fluxo

Descrição: Estruturas de controle como `if`, `for` e `while` permitem controlar o fluxo de execução.

Exemplo 1: Condicionais com `if`.

#!/bin/bash
idade=18
if [ $idade -ge 18 ]; then
    echo "Maior de idade"
else
    echo "Menor de idade"
fi

Exemplo 2: Laços com `for`.

#!/bin/bash
for i in {1..5}; do
    echo "Número: $i"
done
4. Funções

Descrição: Funções são blocos de código reutilizáveis que podem aceitar parâmetros e retornar valores.

Exemplo 1: Função básica.

#!/bin/bash
saudacao() {
    echo "Olá, $1!"
}
saudacao "João"

Exemplo 2: Função com retorno.

#!/bin/bash
soma() {
    local resultado=$(( $1 + $2 ))
    echo $resultado
}
resultado=$(soma 5 3)
echo "A soma é: $resultado"
5. Manipulação de Arquivos

Descrição: Shell Script permite manipular arquivos e diretórios facilmente.

Exemplo 1: Criando e lendo arquivos.

#!/bin/bash
echo "Este é um conteúdo de exemplo" > arquivo.txt
cat arquivo.txt

Exemplo 2: Verificando se um arquivo existe.

#!/bin/bash
if [ -f "arquivo.txt" ]; then
    echo "O arquivo existe!"
else
    echo "O arquivo não existe."
fi
6. Comandos Úteis

Descrição: Alguns comandos úteis para manipulação de arquivos, processos e sistema.

Exemplo 1: Listando arquivos.

#!/bin/bash
ls -l

Exemplo 2: Monitorando processos.

#!/bin/bash
ps aux | grep bash
7. Parâmetros de Linha de Comando

Descrição: Scripts podem receber parâmetros via linha de comando.

Exemplo 1: Recebendo parâmetros.

#!/bin/bash
echo "Primeiro parâmetro: $1"
echo "Segundo parâmetro: $2"

Exemplo 2: Contando parâmetros.

#!/bin/bash
echo "Número de parâmetros: $#"
8. Entrada de Usuário

Descrição: Solicitar entrada do usuário durante a execução do script.

Exemplo 1: Lendo entrada do usuário.

#!/bin/bash
echo "Qual é o seu nome?"
read nome
echo "Olá, $nome!"

Exemplo 2: Entrada silenciosa (sem eco).

#!/bin/bash
read -s -p "Digite sua senha: " senha
echo ""
echo "Senha recebida!"
9. Arrays

Descrição: Arrays são coleções ordenadas de elementos.

Exemplo 1: Declaração e acesso a arrays.

#!/bin/bash
frutas=("Maçã" "Banana" "Laranja")
echo "Primeira fruta: ${frutas[0]}"
echo "Todas as frutas: ${frutas[@]}"

Exemplo 2: Iterando sobre arrays.

#!/bin/bash
frutas=("Maçã" "Banana" "Laranja")
for fruta in "${frutas[@]}"; do
    echo "Fruta: $fruta"
done
10. Tratamento de Erros

Descrição: Tratar erros e verificar o status de comandos.

Exemplo 1: Verificando o status de saída.

#!/bin/bash
ls arquivo_inexistente
if [ $? -ne 0 ]; then
    echo "Erro ao listar o arquivo."
fi

Exemplo 2: Tratamento de erro com `set -e`.

#!/bin/bash
set -e
comando_invalido
echo "Este comando nunca será executado."
11. Automatização de Tarefas

Descrição: Shell Script é ideal para automatizar tarefas repetitivas.

Exemplo 1: Backup automático.

#!/bin/bash
data=$(date +%Y-%m-%d)
cp -r /caminho/origem /caminho/destino/backup-$data
echo "Backup concluído em backup-$data"

Exemplo 2: Monitoramento de disco.

#!/bin/bash
uso=$(df / | grep / | awk '{print $5}' | sed 's/%//g')
if [ $uso -gt 80 ]; then
    echo "Atenção: Disco cheio ($uso%)!"
fi
12. Melhores Práticas

Descrição: Boas práticas incluem uso de shebang, modularização e tratamento de erros.

Exemplo 1: Shebang correto.

#!/bin/bash
# Sempre use o shebang correto no início do script

Exemplo 2: Modularização do código.

#!/bin/bash
# Função separada
verificar_espaco_disco() {
    uso=$(df / | grep / | awk '{print $5}' | sed 's/%//g')
    if [ $uso -gt 80 ]; then
        echo "Atenção: Disco cheio ($uso%)!"
    fi
}

# Chamada da função
verificar_espaco_disco
