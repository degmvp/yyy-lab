Guia JavaScript

1. Introdução à Linguagem

Descrição: JavaScript é uma linguagem de programação versátil usada principalmente para desenvolvimento web. Ela é executada no navegador e também no backend com Node.js.

Exemplo 1: Primeiro programa em JavaScript.

console.log("Olá, mundo!");

Exemplo 2: Inserindo JavaScript em HTML.
2. Fundamentos

Descrição: Os fundamentos incluem variáveis, tipos de dados e operadores.

Exemplo 1: Declaração de variáveis.

let nome = "JavaScript";
const anoCriacao = 1995;
var versao = "ES6"; // Evite usar var
console.log(nome, anoCriacao, versao);

Exemplo 2: Tipos de dados.

let numero = 42; // Number
let texto = "Olá"; // String
let ativo = true; // Boolean
let lista = [1, 2, 3]; // Array
let objeto = { chave: "valor" }; // Object
console.log(typeof numero, typeof texto, typeof ativo, typeof lista, typeof objeto);
3. Operadores

Descrição: JavaScript suporta operadores aritméticos, relacionais e lógicos.

Exemplo 1: Operadores aritméticos.

let a = 10;
let b = 3;
console.log("Soma:", a + b); // 13
console.log("Subtração:", a - b); // 7
console.log("Multiplicação:", a * b); // 30
console.log("Divisão:", a / b); // 3.333...
console.log("Resto:", a % b); // 1

Exemplo 2: Operadores lógicos.

let x = true;
let y = false;
console.log("AND:", x && y); // false
console.log("OR:", x || y); // true
console.log("NOT:", !x); // false
4. Controle de Fluxo

Descrição: Estruturas de controle como `if`, `for` e `while` permitem controlar o fluxo de execução.

Exemplo 1: Condicionais com `if`.

let idade = 18;
if (idade >= 18) {
    console.log("Maior de idade");
} else {
    console.log("Menor de idade");
}

Exemplo 2: Laços com `for`.

for (let i = 1; i <= 5; i++) {
    console.log("Número:", i);
}
5. Funções

Descrição: Funções são blocos de código reutilizáveis que podem aceitar parâmetros e retornar valores.

Exemplo 1: Função básica.

function saudacao(nome) {
    return `Olá, ${nome}!`;
}
console.log(saudacao("João"));

Exemplo 2: Funções anônimas e arrow functions.

// Função anônima
const soma = function(a, b) {
    return a + b;
};
console.log(soma(5, 3));

// Arrow function
const multiplicacao = (a, b) => a * b;
console.log(multiplicacao(4, 5));
6. Arrays

Descrição: Arrays são coleções ordenadas de elementos.

Exemplo 1: Manipulação básica de arrays.

let numeros = [1, 2, 3, 4];
numeros.push(5); // Adiciona ao final
numeros.pop(); // Remove o último elemento
console.log(numeros);

// Iterando com forEach
numeros.forEach((numero) => {
    console.log(numero);
});

Exemplo 2: Métodos avançados de array.

let numeros = [1, 2, 3, 4];

// Map
let dobrados = numeros.map((numero) => numero * 2);
console.log(dobrados);

// Filter
let pares = numeros.filter((numero) => numero % 2 === 0);
console.log(pares);

// Reduce
let soma = numeros.reduce((total, numero) => total + numero, 0);
console.log(soma);
7. Objetos

Descrição: Objetos são coleções de pares chave-valor.

Exemplo 1: Criação e manipulação de objetos.

let pessoa = {
    nome: "Maria",
    idade: 25,
    saudacao: function() {
        console.log(`Olá, meu nome é ${this.nome}`);
    }
};
pessoa.saudacao();

Exemplo 2: Acesso dinâmico a propriedades.

let produto = {
    nome: "Camiseta",
    preco: 50
};
console.log(produto["nome"]); // Camiseta
produto["cor"] = "Azul";
console.log(produto.cor); // Azul
8. DOM Manipulation

Descrição: O DOM (Document Object Model) permite interagir com o HTML e CSS usando JavaScript.

Exemplo 1: Alterando conteúdo de um elemento.

document.getElementById("titulo").innerText = "Novo Título";

Exemplo 2: Adicionando eventos.

document.getElementById("botao").addEventListener("click", () => {
    alert("Botão clicado!");
});
9. Promises e Async/Await

Descrição: Promises e async/await facilitam o trabalho com código assíncrono.

Exemplo 1: Uso de Promises.

function obterDados() {
    return new Promise((resolve, reject) => {
        setTimeout(() => resolve("Dados obtidos"), 1000);
    });
}
obterDados().then((resultado) => {
    console.log(resultado);
});

Exemplo 2: Uso de async/await.

async function buscarDados() {
    let resultado = await obterDados();
    console.log(resultado);
}
buscarDados();
10. Classes

Descrição: Classes são uma forma de criar objetos com métodos e propriedades.

Exemplo 1: Criando uma classe.

class Pessoa {
    constructor(nome, idade) {
        this.nome = nome;
        this.idade = idade;
    }
    saudacao() {
        console.log(`Olá, sou ${this.nome} e tenho ${this.idade} anos.`);
    }
}
let joao = new Pessoa("João", 30);
joao.saudacao();

Exemplo 2: Herança de classes.

class Funcionario extends Pessoa {
    constructor(nome, idade, cargo) {
        super(nome, idade);
        this.cargo = cargo;
    }
    detalhes() {
        console.log(`${this.nome} é ${this.cargo}.`);
    }
}
let maria = new Funcionario("Maria", 25, "Desenvolvedora");
maria.detalhes();
11. Módulos

Descrição: Módulos permitem organizar o código em arquivos separados.

Exemplo 1: Exportando e importando módulos.

// math.js
export function soma(a, b) {
    return a + b;
}

// main.js
import { soma } from './math.js';
console.log(soma(5, 3)); // 8
12. Melhores Práticas

Descrição: Boas práticas incluem uso de const/let, evitar var, modularização e tratamento de erros.

Exemplo 1: Tratamento de erros com try/catch.

try {
    throw new Error("Erro simulado");
} catch (erro) {
    console.error("Erro capturado:", erro.message);
}

Exemplo 2: Modularização do código.

// utils.js
export function formatarData(data) {
    return data.toISOString().split("T")[0];
}

// app.js
import { formatarData } from './utils.js';
console.log(formatarData(new Date())); // YYYY-MM-DD
