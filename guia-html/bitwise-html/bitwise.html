Guia Operações Binárias

1. Introdução às Operações Binárias

Descrição: Operações binárias manipulam bits diretamente, usando operadores como AND, OR, XOR, NOT e deslocamentos de bits.

Exemplo 1: Representação binária de números.

Decimal: 13 → Binário: 1101
Decimal: 7  → Binário: 0111

Exemplo 2: Operador AND (&).

13 & 7 = 5
// Explicação:
// 1101 (13)
// 0111 (7)
// ---- AND
// 0101 (5)
2. Operadores Lógicos Bitwise

Descrição: Os principais operadores bitwise são AND (&), OR (|), XOR (^) e NOT (~).

Exemplo 1: Operador OR (|).

13 | 7 = 15
// Explicação:
// 1101 (13)
// 0111 (7)
// ---- OR
// 1111 (15)

Exemplo 2: Operador XOR (^).

13 ^ 7 = 10
// Explicação:
// 1101 (13)
// 0111 (7)
// ---- XOR
// 1010 (10)
3. Conversões de Base

Descrição: Números podem ser convertidos entre bases decimais, binárias e hexadecimais.

Exemplo 1: Decimal → Binário.

Decimal: 25 → Binário: 11001
// Método: Divida por 2 até o quociente ser zero.

Exemplo 2: Binário → Hexadecimal.

Binário: 11001 → Hexadecimal: 19
// Agrupe em blocos de 4 bits: 0001 1001 → 19
4. Arrays e Vetores com Representação Hexadecimal

Descrição: Arrays podem armazenar valores em diferentes bases, incluindo hexadecimal.

Exemplo 1: Array de números hexadecimais.

let vetor = [0x1A, 0xFF, 0x7F];
console.log(vetor); // [26, 255, 127]

Exemplo 2: Manipulação de bits em arrays.

let vetor = [0b1101, 0b1010]; // [13, 10]
vetor[0] = vetor[0] & 0b1010; // 13 & 10 = 8
console.log(vetor); // [8, 10]
5. Aplicações Práticas: Bitmasking

Descrição: Bitmasking é uma técnica usada para manipular ou verificar bits específicos em um número.

Exemplo 1: Verificar se um bit está ativo.

let numero = 0b1010; // 10 em decimal
let mascara = 0b0010; // Verifica o segundo bit
if (numero & mascara) {
    console.log("O segundo bit está ativo.");
} else {
    console.log("O segundo bit está inativo.");
}

Exemplo 2: Ativar ou desativar bits.

let numero = 0b1010; // 10 em decimal
numero |= 0b0100;     // Ativa o terceiro bit → 1110 (14)
numero &= ~0b0010;    // Desativa o segundo bit → 1100 (12)
console.log(numero.toString(2)); // "1100"
6. Conversão Entre Bases

Descrição: Conversões entre bases são úteis em sistemas de baixo nível, como redes e criptografia.

Exemplo 1: Decimal → Hexadecimal.

Decimal: 255 → Hexadecimal: FF
// Método: Divida por 16 até o quociente ser zero.

Exemplo 2: Hexadecimal → Binário.

Hexadecimal: 1A → Binário: 00011010
// Converta cada dígito hexadecimal para 4 bits.
7. Algoritmos com Bitwise

Descrição: Algoritmos que usam bitwise são rápidos e eficientes em termos de memória.

Exemplo 1: Contagem de bits ativos (Hamming Weight).

function contarBits(n) {
    let count = 0;
    while (n) {
        count += n & 1; // Verifica o bit menos significativo
        n >>= 1;        // Desloca para a direita
    }
    return count;
}
console.log(contarBits(13)); // 1101 → 3 bits ativos

Exemplo 2: Troca de valores sem variável temporária.

let a = 5, b = 7;
a = a ^ b; // a = 12 (1100)
b = a ^ b; // b = 5  (0101)
a = a ^ b; // a = 7  (0111)
console.log(a, b); // 7, 5
8. Exemplos Práticos de Bitmasking

Descrição: Bitmasking é amplamente usado em flags, permissões e otimizações.

Exemplo 1: Gerenciamento de permissões.

const PERMISSAO_LEITURA = 0b0001;
const PERMISSAO_ESCRITA = 0b0010;
const PERMISSAO_EXECUCAO = 0b0100;

let usuario = PERMISSAO_LEITURA | PERMISSAO_ESCRITA; // 0b0011
if (usuario & PERMISSAO_LEITURA) {
    console.log("Permissão de leitura concedida.");
}

Exemplo 2: Flags em jogos.

const FLAG_VIVO = 0b0001;
const FLAG_INVENCIVEL = 0b0010;

let estado = FLAG_VIVO | FLAG_INVENCIVEL; // 0b0011
if (estado & FLAG_VIVO) {
    console.log("O jogador está vivo.");
}
