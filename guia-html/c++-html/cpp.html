Guia de Referência C++

1. Variáveis e Tipos de Dados

Variáveis armazenam dados na memória, e cada variável possui um tipo que define o tipo de dado que pode conter (e.g., int, float, double, char, bool). C++ é uma linguagem fortemente tipada, exigindo declaração explícita do tipo. Escopo (local ou global) e tempo de vida também são importantes.

1. Variáveis e Tipos de Dados - Exemplo 1

Declaração e inicialização de variáveis


#include 
using namespace std;
int main() {
    int idade = 25; // Inteiro
    float altura = 1.75; // Ponto flutuante
    cout << "Idade: " << idade << ", Altura: " << altura << endl;
    return 0;
}

1. Variáveis e Tipos de Dados - Exemplo 2

Variável com escopo local


#include 
using namespace std;
void funcao() {
    int temp = 10; // Só existe dentro da função
    cout << "Temp: " << temp << endl;
}
int main() {
    funcao();
    return 0;
}

1. Variáveis e Tipos de Dados - Detalhes Adicionais

Use const para variáveis imutáveis (e.g., const int MAX = 100;). Tipos como unsigned int ou long long são úteis para números não negativos ou muito grandes.

2. Entrada e Saída (I/O)

A biblioteca permite entrada (cin) e saída (cout) de dados. Use cout para exibir informações no console e cin para capturar entrada do usuário. Os operadores << e >> formatam a entrada/saída.

2. Entrada e Saída (I/O) - Exemplo 1

Saída simples


#include 
using namespace std;
int main() {
    cout << "Bem-vindo ao C++!" << endl;
    return 0;
}

2. Entrada e Saída (I/O) - Exemplo 2

Entrada e saída combinadas


#include 
#include 
using namespace std;
int main() {
    string nome;
    int idade;
    cout << "Digite seu nome e idade: ";
    cin >> nome >> idade;
    cout << "Olá, " << nome << "! Você tem " << idade << " anos." << endl;
    return 0;
}

2. Entrada e Saída (I/O) - Detalhes Adicionais

Use endl para nova linha ou \n para maior eficiência. Para formatar a saída, inclua (e.g., setw, setprecision).

3. Operadores

C++ suporta operadores aritméticos (+, -, *, /, %), relacionais (==, !=, >, <), lógicos (&&, ||, !) e de atribuição (=, +=, etc.). A precedência determina a ordem de avaliação, mas parênteses garantem clareza.

3. Operadores - Exemplo 1

Operadores aritméticos e relacionais


#include 
using namespace std;
int main() {
    int a = 10, b = 3;
    cout << "Soma: " << a + b << ", Divisão: " << a / b << endl;
    cout << "a > b? " << (a > b) << endl; // 1 (verdadeiro)
    return 0;
}

3. Operadores - Exemplo 2

Operadores lógicos


#include 
using namespace std;
int main() {
    bool x = true, y = false;
    if (x && !y) {
        cout << "Condição verdadeira!" << endl;
    }
    return 0;
}

3. Operadores - Detalhes Adicionais

O operador % (módulo) só funciona com inteiros. Use ++ ou -- para incremento/decremento (e.g., i++ ou ++i).

4. Estruturas Condicionais

Estruturas como if, else if, else e switch controlam o fluxo com base em condições. O switch é útil para comparar uma variável contra múltiplos valores constantes, enquanto if é mais flexível.

4. Estruturas Condicionais - Exemplo 1

If-else


#include 
using namespace std;
int main() {
    int nota = 75;
    if (nota >= 90) {
        cout << "Aprovado com distinção!" << endl;
    } else if (nota >= 60) {
        cout << "Aprovado!" << endl;
    } else {
        cout << "Reprovado." << endl;
    }
    return 0;
}

4. Estruturas Condicionais - Exemplo 2

Switch


#include 
using namespace std;
int main() {
    char dia = '2';
    switch (dia) {
        case '1': cout << "Domingo" << endl; break;
        case '2': cout << "Segunda" << endl; break;
        default: cout << "Dia inválido" << endl;
    }
    return 0;
}

4. Estruturas Condicionais - Detalhes Adicionais

Sempre use break em switch para evitar "fall-through". Condições complexas podem combinar operadores lógicos.

5. Laços de Repetição

Laços como for, while e do-while executam blocos de código repetidamente. O for é ideal para iterações com contador, enquanto while e do-while são mais flexíveis para condições gerais.

5. Laços de Repetição - Exemplo 1

Laço for


#include 
using namespace std;
int main() {
    for (int i = 1; i <= 5; i++) {
        cout << "Iteração " << i << endl;
    }
    return 0;
}

5. Laços de Repetição - Exemplo 2

Laço while


#include 
using namespace std;
int main() {
    int num = 10;
    while (num > 0) {
        cout << num << " ";
        num -= 2;
    }
    cout << endl;
    return 0;
}

5. Laços de Repetição - Detalhes Adicionais

Use break para sair de um laço e continue para pular para a próxima iteração. Evite laços infinitos verificando a condição de parada.

6. Funções

Funções dividem o código em blocos reutilizáveis. Elas têm um tipo de retorno (ou void), nome, parâmetros e corpo. Parâmetros podem ser passados por valor ou referência.

6. Funções - Exemplo 1

Função com retorno


#include 
using namespace std;
int quadrado(int x) {
    return x * x;
}
int main() {
    cout << "Quadrado de 5: " << quadrado(5) << endl; // 25
    return 0;
}

6. Funções - Exemplo 2

Função com passagem por referência


#include 
using namespace std;
void trocar(int& a, int& b) {
    int temp = a; a = b; b = temp;
}
int main() {
    int x = 10, y = 20;
    trocar(x, y);
    cout << "x: " << x << ", y: " << y << endl; // x: 20, y: 10
    return 0;
}

6. Funções - Detalhes Adicionais

Funções podem ser sobrecarregadas (mesmo nome, parâmetros diferentes). Declare protótipos no início do programa para organização.

7. Arrays

Arrays armazenam múltiplos elementos do mesmo tipo em posições contíguas. O acesso é feito por índices (a partir de 0). Arrays têm tamanho fixo em C++ padrão, mas podem ser dinâmicos com new.

7. Arrays - Exemplo 1

Array simples


#include 
using namespace std;
int main() {
    int notas[5] = {85, 90, 78, 92, 88};
    cout << "Primeira nota: " << notas[0] << endl;
    return 0;
}

7. Arrays - Exemplo 2

Iterando sobre array


#include 
using namespace std;
int main() {
    int notas[5] = {85, 90, 78, 92, 88};
    for (int i = 0; i < 5; i++) {
        cout << "Nota " << i + 1 << ": " << notas[i] << endl;
    }
    return 0;
}

7. Arrays - Detalhes Adicionais

Arrays fora dos limites causam comportamento indefinido. Para arrays dinâmicos, use delete[] após new.

8. Ponteiros

Ponteiros armazenam endereços de memória. Use * para declarar um ponteiro e & para obter o endereço de uma variável. Ponteiros são úteis para gerenciamento dinâmico de memória e passagem por referência.

8. Ponteiros - Exemplo 1

Ponteiro básico


#include 
using namespace std;
int main() {
    int x = 10;
    int* ptr = &x;
    cout << "Valor de x: " << *ptr << endl; // 10
    return 0;
}

8. Ponteiros - Exemplo 2

Alocação dinâmica


#include 
using namespace std;
int main() {
    int* arr = new int[3];
    arr[0] = 1; arr[1] = 2; arr[2] = 3;
    cout << "Segundo elemento: " << arr[1] << endl;
    delete[] arr; // Libera memória
    return 0;
}

8. Ponteiros - Detalhes Adicionais

Evite ponteiros nulos ou "dangling pointers". Use nullptr em vez de NULL para maior segurança.

9. Strings

A classe string (biblioteca ) facilita manipulação de texto, com métodos como length(), substr(), e find(). Strings em C (char arrays) são menos comuns, mas ainda usadas.

9. Strings - Exemplo 1

Operações com string


#include 
#include 
using namespace std;
int main() {
    string texto = "Olá, mundo!";
    cout << "Tamanho: " << texto.length() << endl;
    cout << "Subtexto: " << texto.substr(0, 3) << endl; // Olá
    return 0;
}

9. Strings - Exemplo 2

Concatenação


#include 
#include 
using namespace std;
int main() {
    string nome = "Ana";
    string saudacao = "Bem-vinda, " + nome + "!";
    cout << saudacao << endl;
    return 0;
}

9. Strings - Detalhes Adicionais

Para entrada com espaços, use getline(cin, str). Strings em C (e.g., char texto[]) exigem para funções como strlen.

10. Estruturas (Structs)

Structs agrupam variáveis relacionadas sob um único nome. São úteis para representar objetos compostos, como registros. Em C++, structs podem incluir métodos, semelhantes a classes.

10. Estruturas (Structs) - Exemplo 1

Struct simples


#include 
#include 
using namespace std;
struct Pessoa {
    string nome;
    int idade;
};
int main() {
    Pessoa p = {"João", 30};
    cout << p.nome << " tem " << p.idade << " anos." << endl;
    return 0;
}

10. Estruturas (Structs) - Exemplo 2

Array de structs


#include 
#include 
using namespace std;
struct Pessoa {
    string nome;
    int idade;
};
int main() {
    Pessoa grupo[2] = {{"Ana", 25}, {"Carlos", 40}};
    for (int i = 0; i < 2; i++) {
        cout << grupo[i].nome << endl;
    }
    return 0;
}

10. Estruturas (Structs) - Detalhes Adicionais

Use typedef para aliases (e.g., typedef struct {...} Pessoa;). Structs são públicas por padrão, ao contrário de classes.

11. Classes e Objetos

Classes definem tipos personalizados com atributos (dados) e métodos (funções). Objetos são instâncias de classes. Encapsulamento é obtido com modificadores como private e public.

11. Classes e Objetos - Exemplo 1

Classe simples


#include 
#include 
using namespace std;
class Carro {
public:
    string marca;
    int velocidade;
    void acelerar() { velocidade += 10; }
};
int main() {
    Carro c = {"Toyota", 100};
    c.acelerar();
    cout << "Velocidade: " << c.velocidade << endl; // 110
    return 0;
}

11. Classes e Objetos - Exemplo 2

Encapsulamento


#include 
using namespace std;
class Conta {
private:
    double saldo;
public:
    Conta(double s) : saldo(s) {}
    void depositar(double valor) { saldo += valor; }
    double getSaldo() { return saldo; }
};
int main() {
    Conta conta(500.0);
    conta.depositar(200.0);
    cout << "Saldo: " << conta.getSaldo() << endl; // 700
    return 0;
}

11. Classes e Objetos - Detalhes Adicionais

Use construtores para inicializar objetos. Métodos getter e setter ajudam no encapsulamento.

12. Gerenciamento de Exceções

Exceções lidam com erros em tempo de execução usando try, catch e throw. Isso permite que o programa trate falhas sem travar, como divisões por zero ou acesso inválido.

12. Gerenciamento de Exceções - Exemplo 1

Exceção simples


#include 
using namespace std;
int main() {
    try {
        int x = 10, y = 0;
        if (y == 0) throw "Divisão por zero!";
        cout << x / y << endl;
    } catch (const char* msg) {
        cout << "Erro: " << msg << endl;
    }
    return 0;
}

12. Gerenciamento de Exceções - Exemplo 2

Exceção com std::exception


#include 
#include 
using namespace std;
int main() {
    try {
        int idade = -5;
        if (idade < 0) throw invalid_argument("Idade inválida!");
        cout << "Idade: " << idade << endl;
    } catch (const invalid_argument& e) {
        cout << "Erro: " << e.what() << endl;
    }
    return 0;
}

12. Gerenciamento de Exceções - Detalhes Adicionais

Use exceções padrão como std::runtime_error ou std::out_of_range. Sempre limpe recursos no bloco catch para evitar vazamentos.
