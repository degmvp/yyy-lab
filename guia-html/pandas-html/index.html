Guia Pandas

1. Introdução à Biblioteca

Descrição: Pandas é uma biblioteca Python para análise e manipulação de dados. É amplamente usada em ciência de dados e aprendizado de máquina.

Exemplo 1: Instalação do Pandas.

# Execute no terminal:
# pip install pandas

Exemplo 2: Importação básica.

import pandas as pd
print("Pandas importado com sucesso!")
2. Fundamentos

Descrição: Os fundamentos incluem DataFrames e Series, as principais estruturas de dados do Pandas.

Exemplo 1: Criando uma Series.

import pandas as pd
s = pd.Series([1, 2, 3, 4])
print(s)

Exemplo 2: Criando um DataFrame.

import pandas as pd
data = {"Nome": ["João", "Maria"], "Idade": [25, 30]}
df = pd.DataFrame(data)
print(df)
3. Conceitos Fundamentais

Descrição: Entenda como trabalhar com índices, colunas e dados ausentes.

Exemplo 1: Índices personalizados.

import pandas as pd
s = pd.Series([10, 20, 30], index=["a", "b", "c"])
print(s["b"])  # Saída: 20

Exemplo 2: Tratamento de dados ausentes.

import pandas as pd
data = {"A": [1, 2, None], "B": [4, None, 6]}
df = pd.DataFrame(data)
print(df.dropna())  # Remove linhas com NaN
4. Tipos de Dados

Descrição: Pandas suporta diferentes tipos de dados, como inteiros, floats, strings e datas.

Exemplo 1: Conversão de tipos.

import pandas as pd
data = {"A": ["1", "2", "3"]}
df = pd.DataFrame(data)
df["A"] = df["A"].astype(int)
print(df.dtypes)  # Saída: A    int32

Exemplo 2: Manipulação de datas.

import pandas as pd
datas = pd.date_range("2023-01-01", periods=5)
print(datas)
5. Operadores

Descrição: Pandas permite operações aritméticas e lógicas em DataFrames e Series.

Exemplo 1: Operações aritméticas.

import pandas as pd
s = pd.Series([1, 2, 3])
print(s + 10)  # Saída: [11, 12, 13]

Exemplo 2: Filtragem com operadores lógicos.

import pandas as pd
df = pd.DataFrame({"A": [1, 2, 3], "B": [4, 5, 6]})
print(df[df["A"] > 1])  # Filtra linhas onde A > 1
6. Memória

Descrição: Pandas oferece métodos para otimização de memória, como categorias e tipos reduzidos.

Exemplo 1: Uso de categorias.

import pandas as pd
data = {"Categoria": ["A", "B", "A", "B"]}
df = pd.DataFrame(data)
df["Categoria"] = df["Categoria"].astype("category")
print(df.memory_usage())

Exemplo 2: Redução de tipos numéricos.

import pandas as pd
df = pd.DataFrame({"A": [1, 2, 3]}, dtype="int8")
print(df.dtypes)  # Saída: A    int8
7. Controle de Fluxo

Descrição: Use loops e condicionais para manipular dados em Pandas.

Exemplo 1: Iteração com `iterrows`.

import pandas as pd
df = pd.DataFrame({"A": [1, 2, 3]})
for index, row in df.iterrows():
    print(row["A"])

Exemplo 2: Aplicação de funções.

import pandas as pd
df = pd.DataFrame({"A": [1, 2, 3]})
df["A"] = df["A"].apply(lambda x: x * 2)
print(df)
8. Operadores Lógicos

Descrição: Use operadores lógicos para filtrar e manipular dados.

Exemplo 1: Combinação de condições.

import pandas as pd
df = pd.DataFrame({"A": [1, 2, 3], "B": [4, 5, 6]})
print(df[(df["A"] > 1) & (df["B"] < 6)])

Exemplo 2: Negação lógica.

import pandas as pd
df = pd.DataFrame({"A": [True, False, True]})
print(df[~df["A"]])  # Filtra valores falsos
9. Arquivos Texto e Binário

Descrição: Pandas suporta leitura e escrita de arquivos CSV, Excel, JSON, etc.

Exemplo 1: Leitura de arquivo CSV.

import pandas as pd
df = pd.read_csv("dados.csv")
print(df.head())

Exemplo 2: Escrita em arquivo Excel.

import pandas as pd
df = pd.DataFrame({"A": [1, 2, 3]})
df.to_excel("dados.xlsx", index=False)
10. Funções

Descrição: Pandas oferece diversas funções para análise de dados, como agregações e estatísticas.

Exemplo 1: Agregação com `groupby`.

import pandas as pd
df = pd.DataFrame({"A": ["foo", "bar", "foo"], "B": [1, 2, 3]})
print(df.groupby("A").sum())

Exemplo 2: Estatísticas descritivas.

import pandas as pd
df = pd.DataFrame({"A": [1, 2, 3]})
print(df.describe())
11. Armazenamento

Descrição: Pandas permite armazenar dados em formatos eficientes, como HDF5 e Parquet.

Exemplo 1: Armazenamento em HDF5.

import pandas as pd

# Criando um DataFrame
df = pd.DataFrame({"A": [1, 2, 3], "B": [4, 5, 6]})

# Salvando em HDF5
df.to_hdf("dados.h5", key="df", mode="w")

# Lendo do HDF5
df_lido = pd.read_hdf("dados.h5", key="df")
print(df_lido)

Exemplo 2: Armazenamento em Parquet.

import pandas as pd

# Criando um DataFrame
df = pd.DataFrame({"A": [1, 2, 3], "B": [4, 5, 6]})

# Salvando em Parquet
df.to_parquet("dados.parquet")

# Lendo do Parquet
df_lido = pd.read_parquet("dados.parquet")
print(df_lido)
12. Programação Orientada a Objetos

Descrição: Pandas é construído sobre classes e objetos, permitindo extensão personalizada.

Exemplo 1: Criação de classe personalizada.

import pandas as pd
class MeuDataFrame(pd.DataFrame):
    def soma_colunas(self):
        return self.sum(axis=1)

df = MeuDataFrame({"A": [1, 2], "B": [3, 4]})
print(df.soma_colunas())

Exemplo 2: Herança de métodos.

import pandas as pd
class DataFrameFiltrado(pd.DataFrame):
    def filtrar_positivos(self):
        return self[self > 0]

df = DataFrameFiltrado({"A": [-1, 2, 3]})
print(df.filtrar_positivos())
