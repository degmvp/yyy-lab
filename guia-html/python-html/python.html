Guia Python

1. Introdução à Linguagem

Descrição: Python é uma linguagem de programação interpretada, de alto nível, criada por Guido van Rossum em 1991. É conhecida por sua sintaxe simples e legibilidade.

Exemplo 1: Programa básico "Hello, World!".

print("Hello, World!")

Exemplo 2: Verificação da versão do Python instalada.

# Execute no terminal:
# python --version
2. Fundamentos

Descrição: Os fundamentos incluem variáveis, tipos de dados e operações básicas.

Exemplo 1: Declaração de variáveis e constantes.

nome = "João"
idade = 30
PI = 3.14  # Constante (por convenção, em maiúsculas)
print(nome, idade, PI)

Exemplo 2: Uso de módulos externos.

import math
print("Raiz quadrada de 16:", math.sqrt(16))
3. Conceitos Fundamentais

Descrição: Python organiza código em módulos e pacotes. A visibilidade é controlada pelo escopo das funções e classes.

Exemplo 1: Organização de módulos.

# Arquivo utils.py
def somar(a, b):
    return a + b

# Arquivo main.py
import utils
print("Soma:", utils.somar(5, 3))

Exemplo 2: Funções públicas e privadas.

# Função pública
def publica():
    print("Função pública")

# Função privada (por convenção, com underline)
def _privada():
    print("Função privada")
4. Tipos de Dados

Descrição: Python suporta tipos primitivos (inteiros, floats, strings, etc.), listas, tuplas, dicionários e conjuntos.

Exemplo 1: Tipos primitivos.

inteiro = 10
decimal = 3.14
texto = "Python é incrível!"
print(inteiro, decimal, texto)

Exemplo 2: Listas e dicionários.

# Lista
numeros = [1, 2, 3]
numeros.append(4)
print("Lista:", numeros)

# Dicionário
pessoa = {"nome": "João", "idade": 30}
print("Dicionário:", pessoa)
5. Operadores

Descrição: Python suporta operadores aritméticos, relacionais e lógicos.

Exemplo 1: Operadores aritméticos.

a = 10
b = 3
print("Soma:", a + b)
print("Subtração:", a - b)
print("Multiplicação:", a * b)
print("Divisão:", a / b)
print("Resto:", a % b)

Exemplo 2: Operadores lógicos.

x = True
y = False
print("AND:", x and y)
print("OR:", x or y)
print("NOT:", not x)
6. Memória

Descrição: Python gerencia memória automaticamente com garbage collection, mas também permite controle manual usando referências.

Exemplo 1: Uso de referências.

x = [1, 2, 3]
y = x  # y aponta para o mesmo objeto que x
y[0] = 10
print("x:", x)  # Saída: [10, 2, 3]

Exemplo 2: Cópia de objetos.

import copy
x = [1, 2, 3]
y = copy.deepcopy(x)  # Faz uma cópia profunda
y[0] = 10
print("x:", x)  # Saída: [1, 2, 3]
7. Controle de Fluxo

Descrição: Estruturas de controle como `if`, `for` e `while` permitem controlar o fluxo de execução.

Exemplo 1: Condicionais com `if`.

idade = 18
if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")

Exemplo 2: Laços com `for`.

for i in range(1, 6):
    print("Número:", i)
8. Operadores Lógicos

Descrição: Operadores lógicos (`and`, `or`, `not`) são usados para combinar condições.

Exemplo 1: Combinação de condições.

x = 10
y = 20
if x > 5 and y < 30:
    print("Ambas as condições são verdadeiras")

Exemplo 2: Negação lógica.

ativo = False
if not ativo:
    print("Inativo")
9. Arquivos Texto e Binário

Descrição: Python oferece suporte para leitura e escrita de arquivos texto e binários.

Exemplo 1: Leitura de arquivo texto.

with open("arquivo.txt", "r") as arquivo:
    conteudo = arquivo.read()
    print(conteudo)

Exemplo 2: Escrita em arquivo binário.

with open("arquivo.bin", "wb") as arquivo:
    arquivo.write(b"\x00\xFF")
10. Funções

Descrição: Funções são blocos de código reutilizáveis que podem aceitar parâmetros e retornar valores.

Exemplo 1: Função com parâmetros e retorno.

def soma(a, b):
    return a + b

print("Soma:", soma(5, 3))

Exemplo 2: Função com múltiplos retornos.

def dividir(a, b):
    if b == 0:
        return None, "Divisão por zero"
    return a / b, None

resultado, erro = dividir(10, 2)
if erro:
    print("Erro:", erro)
else:
    print("Resultado:", resultado)
11. Armazenamento

Descrição: Python suporta armazenamento em bancos de dados SQL e NoSQL, além de manipulação de dados em memória.

Exemplo 1: Conexão com SQLite.

import sqlite3
conexao = sqlite3.connect("database.db")
cursor = conexao.cursor()
cursor.execute("CREATE TABLE IF NOT EXISTS usuarios (id INTEGER PRIMARY KEY, nome TEXT)")
conexao.commit()
conexao.close()

Exemplo 2: Armazenamento em memória com dicionários.

cache = {}
cache["chave1"] = 42
print("Valor:", cache["chave1"])
12. Programação Orientada a Objetos

Descrição: Python é uma linguagem orientada a objetos, com suporte a classes, herança e polimorfismo.

Exemplo 1: Classes e métodos.

class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def apresentar(self):
        print(f"Olá, meu nome é {self.nome} e tenho {self.idade} anos.")

pessoa = Pessoa("João", 30)
pessoa.apresentar()

Exemplo 2: Herança e polimorfismo.

class Animal:
    def falar(self):
        pass

class Cachorro(Animal):
    def falar(self):
        return "Au au!"

class Gato(Animal):
    def falar(self):
        return "Miau!"

animais = [Cachorro(), Gato()]
for animal in animais:
    print(animal.falar())
