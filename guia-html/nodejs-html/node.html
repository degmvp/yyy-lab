Guia Node.js

1. Introdução à Linguagem

Descrição: Node.js é uma plataforma JavaScript baseada no motor V8 do Google Chrome. É usada para criar aplicativos backend, APIs e ferramentas CLI.

Exemplo 1: Instalação do Node.js.

# Execute no terminal:
# npm install -g node

Exemplo 2: Verificação da versão instalada.

// Execute no terminal:
// node --version
2. Fundamentos

Descrição: Os fundamentos incluem variáveis, funções e módulos.

Exemplo 1: Variáveis e funções.

const nome = "Node.js";
function saudacao() {
    console.log(`Olá, ${nome}!`);
}
saudacao();

Exemplo 2: Importação de módulos.

const fs = require("fs");
fs.readFile("arquivo.txt", "utf8", (err, data) => {
    if (err) throw err;
    console.log(data);
});
3. Conceitos Fundamentais

Descrição: Node.js usa um modelo assíncrono e não bloqueante para lidar com operações de I/O.

Exemplo 1: Função assíncrona com `setTimeout`.

console.log("Início");
setTimeout(() => {
    console.log("Executando após 2 segundos");
}, 2000);
console.log("Fim");

Exemplo 2: Uso de Promises.

const promise = new Promise((resolve, reject) => {
    setTimeout(() => resolve("Sucesso!"), 1000);
});
promise.then((resultado) => {
    console.log(resultado);
});
4. Tipos de Dados

Descrição: Node.js suporta tipos primitivos como strings, números, booleanos, além de objetos e arrays.

Exemplo 1: Tipos primitivos.

const texto = "Hello, Node.js";
const numero = 42;
const booleano = true;
console.log(texto, numero, booleano);

Exemplo 2: Objetos e arrays.

const pessoa = { nome: "João", idade: 30 };
const numeros = [1, 2, 3, 4];
console.log(pessoa, numeros);
5. Operadores

Descrição: Node.js suporta operadores aritméticos, relacionais e lógicos.

Exemplo 1: Operadores aritméticos.

const a = 10;
const b = 3;
console.log("Soma:", a + b);
console.log("Subtração:", a - b);
console.log("Multiplicação:", a * b);
console.log("Divisão:", a / b);
console.log("Resto:", a % b);

Exemplo 2: Operadores lógicos.

const x = true;
const y = false;
console.log("AND:", x && y);
console.log("OR:", x || y);
console.log("NOT:", !x);
6. Memória

Descrição: Node.js gerencia memória automaticamente, mas permite controle manual usando referências e garbage collection.

Exemplo 1: Uso de referências.

const obj1 = { valor: 42 };
const obj2 = obj1; // obj2 aponta para o mesmo objeto que obj1
obj2.valor = 100;
console.log(obj1.valor); // Saída: 100

Exemplo 2: Limpeza de memória.

let grandeObjeto = new Array(1000000).fill("dados");
grandeObjeto = null; // Libera a memória
7. Controle de Fluxo

Descrição: Estruturas de controle como `if`, `for` e `while` permitem controlar o fluxo de execução.

Exemplo 1: Condicionais com `if`.

const idade = 18;
if (idade >= 18) {
    console.log("Maior de idade");
} else {
    console.log("Menor de idade");
}

Exemplo 2: Laços com `for`.

for (let i = 1; i <= 5; i++) {
    console.log("Número:", i);
}
8. Operadores Lógicos

Descrição: Operadores lógicos (`&&`, `||`, `!`) são usados para combinar condições.

Exemplo 1: Combinação de condições.

const x = 10;
const y = 20;
if (x > 5 && y < 30) {
    console.log("Ambas as condições são verdadeiras");
}

Exemplo 2: Negação lógica.

const ativo = false;
if (!ativo) {
    console.log("Inativo");
}
9. Arquivos Texto e Binário

Descrição: Node.js oferece suporte para leitura e escrita de arquivos texto e binários.

Exemplo 1: Leitura de arquivo texto.

const fs = require("fs");
fs.readFile("arquivo.txt", "utf8", (err, data) => {
    if (err) throw err;
    console.log(data);
});

Exemplo 2: Escrita em arquivo binário.

const fs = require("fs");
const buffer = Buffer.from([0x00, 0xFF]);
fs.writeFile("arquivo.bin", buffer, (err) => {
    if (err) throw err;
    console.log("Arquivo binário criado!");
});
10. Funções

Descrição: Funções são blocos de código reutilizáveis que podem aceitar parâmetros e retornar valores.

Exemplo 1: Função básica.

function soma(a, b) {
    return a + b;
}
console.log("Soma:", soma(5, 3));

Exemplo 2: Função assíncrona.

async function fetchData() {
    const response = await fetch("https://jsonplaceholder.typicode.com/posts/1");
    const data = await response.json();
    console.log(data);
}
fetchData();
11. Armazenamento

Descrição: Node.js pode integrar-se com bancos de dados SQL e NoSQL.

Exemplo 1: Conexão com MongoDB.

const { MongoClient } = require("mongodb");
async function main() {
    const uri = "mongodb://localhost:27017";
    const client = new MongoClient(uri);
    try {
        await client.connect();
        console.log("Conectado ao MongoDB!");
    } finally {
        await client.close();
    }
}
main().catch(console.error);

Exemplo 2: Armazenamento em memória com objetos.

const cache = {};
cache["chave1"] = 42;
console.log("Valor:", cache["chave1"]);
12. Servidor HTTP

Descrição: Node.js é amplamente usado para criar servidores HTTP.

Exemplo 1: Servidor HTTP básico.

const http = require("http");
const server = http.createServer((req, res) => {
    res.writeHead(200, { "Content-Type": "text/plain" });
    res.end("Hello, World!");
});
server.listen(3000, () => {
    console.log("Servidor rodando em http://localhost:3000");
});

Exemplo 2: Roteamento básico.

const http = require("http");
const server = http.createServer((req, res) => {
    if (req.url === "/") {
        res.writeHead(200, { "Content-Type": "text/plain" });
        res.end("Página inicial");
    } else if (req.url === "/sobre") {
        res.writeHead(200, { "Content-Type": "text/plain" });
        res.end("Sobre nós");
    } else {
        res.writeHead(404, { "Content-Type": "text/plain" });
        res.end("Página não encontrada");
    }
});
server.listen(3000, () => {
    console.log("Servidor rodando em http://localhost:3000");
});
