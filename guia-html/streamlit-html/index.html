Guia Streamlit

1. Introdução à Biblioteca

Descrição: Streamlit é uma biblioteca Python para criar aplicativos web interativos rapidamente. É ideal para ciência de dados e machine learning.

Exemplo 1: Instalação do Streamlit.

# Execute no terminal:
# pip install streamlit

Exemplo 2: Execução de um app básico.

# Crie um arquivo app.py com o seguinte conteúdo:
import streamlit as st
st.write("Hello, Streamlit!")

# Execute no terminal:
# streamlit run app.py
2. Fundamentos

Descrição: Os fundamentos incluem exibição de texto, números e dados simples.

Exemplo 1: Exibição de texto e números.

import streamlit as st
st.title("Meu Primeiro App")
st.header("Seção 1")
st.subheader("Subseção 1.1")
st.write("Este é um exemplo de texto.")
st.write(42)  # Exibe um número

Exemplo 2: Exibição de dados tabulares.

import streamlit as st
import pandas as pd
data = {"Nome": ["João", "Maria"], "Idade": [25, 30]}
df = pd.DataFrame(data)
st.write(df)
3. Conceitos Fundamentais

Descrição: Entenda como usar widgets interativos e layout personalizado.

Exemplo 1: Widgets básicos.

import streamlit as st
nome = st.text_input("Digite seu nome:")
idade = st.slider("Escolha sua idade:", 0, 100, 25)
st.write(f"Olá, {nome}! Você tem {idade} anos.")

Exemplo 2: Layout com colunas.

import streamlit as st
col1, col2 = st.columns(2)
with col1:
    st.write("Coluna 1")
with col2:
    st.write("Coluna 2")
4. Tipos de Dados

Descrição: Streamlit suporta diferentes tipos de dados, como texto, números, gráficos e tabelas.

Exemplo 1: Exibição de gráficos.

import streamlit as st
import matplotlib.pyplot as plt
plt.plot([1, 2, 3], [4, 5, 6])
st.pyplot(plt)

Exemplo 2: Upload de arquivos.

import streamlit as st
arquivo = st.file_uploader("Carregue um arquivo CSV")
if arquivo:
    st.write("Arquivo carregado!")
5. Operadores

Descrição: Use operações condicionais e lógicas para criar interatividade.

Exemplo 1: Condições com botões.

import streamlit as st
if st.button("Clique aqui"):
    st.write("Botão clicado!")
else:
    st.write("Aguardando clique...")

Exemplo 2: Combinação de condições.

import streamlit as st
opcao = st.selectbox("Escolha uma opção:", ["Opção 1", "Opção 2"])
if opcao == "Opção 1":
    st.write("Você escolheu Opção 1")
else:
    st.write("Você escolheu Opção 2")
6. Memória

Descrição: Streamlit não mantém estado entre execuções. Use `st.session_state` para persistir dados.

Exemplo 1: Uso de `st.session_state`.

import streamlit as st
if "contador" not in st.session_state:
    st.session_state.contador = 0
if st.button("Incrementar"):
    st.session_state.contador += 1
st.write("Contador:", st.session_state.contador)

Exemplo 2: Reset de estado.

import streamlit as st
if st.button("Resetar"):
    st.session_state.clear()
st.write("Estado atual:", st.session_state)
7. Controle de Fluxo

Descrição: Use controle de fluxo para criar aplicativos dinâmicos.

Exemplo 1: Uso de `st.stop()`.

import streamlit as st
if st.button("Parar execução"):
    st.write("Execução parada!")
    st.stop()
st.write("Esta linha só será executada se o botão não for clicado.")

Exemplo 2: Loop com widgets.

import streamlit as st
for i in range(5):
    st.write(f"Linha {i}")
    if st.button(f"Parar em {i}"):
        break
8. Operadores Lógicos

Descrição: Use operadores lógicos para criar interatividade avançada.

Exemplo 1: Combinação de condições.

import streamlit as st
cond1 = st.checkbox("Condição 1")
cond2 = st.checkbox("Condição 2")
if cond1 and cond2:
    st.write("Ambas as condições são verdadeiras!")
elif cond1 or cond2:
    st.write("Pelo menos uma condição é verdadeira.")
else:
    st.write("Nenhuma condição é verdadeira.")

Exemplo 2: Negação lógica.

import streamlit as st
ativo = st.checkbox("Ativo")
if not ativo:
    st.write("Inativo")
else:
    st.write("Ativo")
9. Arquivos Texto e Binário

Descrição: Streamlit permite upload e download de arquivos.

Exemplo 1: Upload de arquivo.

import streamlit as st
arquivo = st.file_uploader("Carregue um arquivo")
if arquivo:
    st.write("Arquivo carregado!")

Exemplo 2: Download de arquivo.

import streamlit as st
texto = "Conteúdo do arquivo"
st.download_button("Baixar arquivo", texto)
10. Funções

Descrição: Streamlit permite criar funções reutilizáveis para organizar o código.

Exemplo 1: Função para exibir dados.

import streamlit as st
def exibir_dados(dados):
    st.write("Dados recebidos:", dados)

dados = {"A": [1, 2, 3], "B": [4, 5, 6]}
exibir_dados(dados)

Exemplo 2: Função com widgets.

import streamlit as st
def calcular_soma(a, b):
    return a + b

a = st.number_input("Número 1")
b = st.number_input("Número 2")
st.write("Soma:", calcular_soma(a, b))
11. Armazenamento

Descrição: Streamlit pode integrar-se com bancos de dados e APIs externas.

Exemplo 1: Conexão com SQLite.

import streamlit as st
import sqlite3
conexao = sqlite3.connect("database.db")
cursor = conexao.cursor()
cursor.execute("CREATE TABLE IF NOT EXISTS usuarios (id INTEGER PRIMARY KEY, nome TEXT)")
conexao.commit()
conexao.close()
st.write("Banco de dados criado!")

Exemplo 2: Chamada de API.

import streamlit as st
import requests
response = requests.get("https://jsonplaceholder.typicode.com/posts")
st.write(response.json())
12. Programação Orientada a Objetos

Descrição: Streamlit pode ser combinado com classes e objetos para criar aplicativos modulares.

Exemplo 1: Classe para gerenciar dados.

import streamlit as st
class GerenciadorDados:
    def __init__(self):
        self.dados = []

    def adicionar(self, valor):
        self.dados.append(valor)

    def exibir(self):
        st.write("Dados:", self.dados)

gerenciador = GerenciadorDados()
valor = st.text_input("Adicionar dado:")
if st.button("Adicionar"):
    gerenciador.adicionar(valor)
gerenciador.exibir()

Exemplo 2: Herança de classes.

import streamlit as st
class BaseApp:
    def exibir(self):
        st.write("Aplicativo base")

class AppPersonalizado(BaseApp):
    def exibir(self):
        st.write("Aplicativo personalizado")

app = AppPersonalizado()
app.exibir()
