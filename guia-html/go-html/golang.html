Guia Golang

1. Introdução à Linguagem

Descrição: Go (ou Golang) foi criada pelo Google em 2007 por Robert Griesemer, Rob Pike e Ken Thompson. É uma linguagem compilada, de tipagem estática, projetada para ser simples, eficiente e fácil de manter.

Exemplo 1: Programa básico "Hello, World!".

package main
import "fmt"
func main() {
    fmt.Println("Hello, World!")
}

Exemplo 2: Verificação da versão do Go instalada.

// Execute no terminal:
// go version
2. Fundamentos

Descrição: Os fundamentos incluem a estrutura básica de um programa Go, como declarar variáveis, constantes e usar pacotes.

Exemplo 1: Declaração de variáveis e constantes.

package main
import "fmt"
func main() {
    var nome string = "João"
    idade := 30
    const anoAtual = 2023
    fmt.Println("Nome:", nome, "Idade:", idade, "Ano:", anoAtual)
}

Exemplo 2: Uso de pacotes externos.

package main
import (
    "fmt"
    "math"
)
func main() {
    fmt.Println("Raiz quadrada de 16:", math.Sqrt(16))
}
3. Conceitos Fundamentais

Descrição: Go organiza código em pacotes e usa visibilidade baseada em maiúsculas/minúsculas para controlar o acesso.

Exemplo 1: Organização de pacotes.

// Arquivo utils.go
package utils
func Somar(a, b int) int {
    return a + b
}

// Arquivo main.go
package main
import (
    "fmt"
    "./utils"
)
func main() {
    fmt.Println("Soma:", utils.Somar(5, 3))
}

Exemplo 2: Visibilidade de funções.

// Função pública (exportada)
func Publica() {
    fmt.Println("Função pública")
}

// Função privada (não exportada)
func privada() {
    fmt.Println("Função privada")
}
4. Tipos de Dados

Descrição: Go suporta tipos primitivos (inteiros, floats, strings, etc.), arrays, slices, structs e mapas.

Exemplo 1: Tipos primitivos.

package main
import "fmt"
func main() {
    var inteiro int = 10
    var decimal float64 = 3.14
    var texto string = "Go é incrível!"
    fmt.Println(inteiro, decimal, texto)
}

Exemplo 2: Arrays e slices.

package main
import "fmt"
func main() {
    // Array fixo
    numeros := [3]int{1, 2, 3}
    fmt.Println("Array:", numeros)

    // Slice dinâmico
    slice := []int{4, 5, 6}
    slice = append(slice, 7)
    fmt.Println("Slice:", slice)
}
5. Operadores

Descrição: Operadores aritméticos, relacionais e lógicos são usados para realizar operações matemáticas e comparações.

Exemplo 1: Operadores aritméticos.

package main
import "fmt"
func main() {
    a := 10
    b := 3
    fmt.Println("Soma:", a+b)
    fmt.Println("Subtração:", a-b)
    fmt.Println("Multiplicação:", a*b)
    fmt.Println("Divisão:", a/b)
    fmt.Println("Resto:", a%b)
}

Exemplo 2: Operadores lógicos.

package main
import "fmt"
func main() {
    x := true
    y := false
    fmt.Println("AND:", x && y)
    fmt.Println("OR:", x || y)
    fmt.Println("NOT:", !x)
}
6. Memória

Descrição: Go gerencia memória automaticamente com garbage collection, mas também permite controle manual usando ponteiros.

Exemplo 1: Uso de ponteiros.

package main
import "fmt"
func main() {
    x := 10
    p := &x
    fmt.Println("Valor de x:", *p)
    *p = 20
    fmt.Println("Novo valor de x:", x)
}

Exemplo 2: Alocação de memória com `new`.

package main
import "fmt"
func main() {
    x := new(int)
    *x = 42
    fmt.Println("Valor:", *x)
}
7. Controle de Fluxo

Descrição: Estruturas de controle como `if`, `switch` e `for` permitem controlar o fluxo de execução.

Exemplo 1: Condicionais com `if`.

package main
import "fmt"
func main() {
    idade := 18
    if idade >= 18 {
        fmt.Println("Maior de idade")
    } else {
        fmt.Println("Menor de idade")
    }
}

Exemplo 2: Laços com `for`.

package main
import "fmt"
func main() {
    for i := 1; i <= 5; i++ {
        fmt.Println("Número:", i)
    }
}
8. Operadores Lógicos

Descrição: Operadores lógicos (`&&`, `||`, `!`) são usados para combinar condições.

Exemplo 1: Combinação de condições.

package main
import "fmt"
func main() {
    x := 10
    y := 20
    if x > 5 && y < 30 {
        fmt.Println("Ambas as condições são verdadeiras")
    }
}

Exemplo 2: Negação lógica.

package main
import "fmt"
func main() {
    ativo := false
    if !ativo {
        fmt.Println("Inativo")
    }
}
9. Arquivos Texto e Binário

Descrição: Go oferece suporte para leitura e escrita de arquivos texto e binários.

Exemplo 1: Leitura de arquivo texto.

package main
import (
    "fmt"
    "io/ioutil"
    "log"
)
func main() {
    data, err := ioutil.ReadFile("arquivo.txt")
    if err != nil {
        log.Fatal(err)
    }
    fmt.Println(string(data))
}

Exemplo 2: Escrita em arquivo binário.

package main
import (
    "encoding/binary"
    "os"
)
func main() {
    file, _ := os.Create("arquivo.bin")
    defer file.Close()
    var numero uint32 = 12345
    binary.Write(file, binary.LittleEndian, numero)
}
10. Funções

Descrição: Funções são blocos de código reutilizáveis que podem aceitar parâmetros e retornar valores.

Exemplo 1: Função com parâmetros e retorno.

package main
import "fmt"
func soma(a, b int) int {
    return a + b
}
func main() {
    fmt.Println("Soma:", soma(5, 3))
}

Exemplo 2: Função com múltiplos retornos.

package main
import "fmt"
func dividir(a, b float64) (float64, error) {
    if b == 0 {
        return 0, fmt.Errorf("divisão por zero")
    }
    return a / b, nil
}
func main() {
    resultado, err := dividir(10, 2)
    if err != nil {
        fmt.Println("Erro:", err)
    } else {
        fmt.Println("Resultado:", resultado)
    }
}
11. Armazenamento

Descrição: Go suporta armazenamento em bancos de dados SQL e NoSQL, além de manipulação de dados em memória.

Exemplo 1: Conexão com SQLite.

package main
import (
    "database/sql"
    "fmt"
    _ "github.com/mattn/go-sqlite3"
)
func main() {
    db, _ := sql.Open("sqlite3", "./database.db")
    defer db.Close()
    _, err := db.Exec("CREATE TABLE IF NOT EXISTS usuarios (id INTEGER PRIMARY KEY, nome TEXT)")
    if err != nil {
        fmt.Println("Erro:", err)
    }
}

Exemplo 2: Armazenamento em memória com mapas.

package main
import "fmt"
func main() {
    cache := make(map[string]int)
    cache["chave1"] = 42
    fmt.Println("Valor:", cache["chave1"])
}
12. Programação Orientada a Objetos

Descrição: Go não tem classes tradicionais, mas sim structs e métodos associados para simular orientação a objetos.

Exemplo 1: Structs e métodos.

package main
import "fmt"
type Pessoa struct {
    Nome  string
    Idade int
}
func (p Pessoa) apresentar() {
    fmt.Printf("Olá, meu nome é %s e tenho %d anos.\n", p.Nome, p.Idade)
}
func main() {
    pessoa := Pessoa{"João", 30}
    pessoa.apresentar()
}

Exemplo 2: Interfaces.

package main
import "fmt"
type Forma interface {
    Area() float64
}
type Retangulo struct {
    Largura, Altura float64
}
func (r Retangulo) Area() float64 {
    return r.Largura * r.Altura
}
func main() {
    var forma Forma = Retangulo{Largura: 5, Altura: 3}
    fmt.Println("Área:", forma.Area())
}
